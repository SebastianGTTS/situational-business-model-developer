Method Composition

The second stage of Method Composition is used by the Method Engineer to compose a development method out of the repositories. The choices are guided by the Situational Factors and the Application Domains. This can be done Pattern-based or Phase-based.

The Pattern-based Composition uses the Method Patterns that are combined and filled with the Method Building Blocks. Here, at the beginning, one Method Pattern with the initialisation type element is select from the repository. Each predefined activity within this pattern can be filled with another Method Pattern or a Method Building Block. During the composition, the development method is checked for correctness.

The Phase-based Composition uses the Phase Elements that are selected and filled with the Method Building Blocks. Here, at the beginning, multiple Phase Elements that like to used used are selected from the repository. Each phase is filled with corresponding Method Building Blocks for the phase. During the composition, the development method is checked for correctness.
