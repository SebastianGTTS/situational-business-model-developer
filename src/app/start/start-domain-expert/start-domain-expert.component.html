Options for the Domain Expert

The following steps are needed to store knowledge about development methods in the Method Repository:

  1. Create Method Elements in the Method Repository
  2. Create Method Building Blocks out of these Method Elements
  3. Create Method Patterns that will be used to connect different Method Building Blocks

 

The following steps are needed to store knowledge about business models in the Canvas Model Repository:

  1. Create Canvas Elements in the Canvas Model Repository
  2. Create Canvas Models to structure and visualize the Canvas Elements
  3. Create Canvas Building Blocks for these Canvas Models
